私、武居紘史が作成しました。
qtというGUIライブラリと、mingwという開発環境で、windows 10 32bit 向けに作成しました
当、[editor.zip]ファイルには、ソースコードとバイナリが組み込まれており、
展開してやることで、利用できる仕組みにしました。
ファイルの構造ですが、

main.cpp -> 本プログラムのソースコード
Makefile -> バイナリ生成自動化ツール make のコード
releaseディレクトリ -> リリースアプリ
debug ディレクトリ -> デバッグアプリ

という形になっています。
このプログラムを実行するには、
[release] ディレクトリ内の、「viewer.exe」という
実行可能ファイルを、選択して、実行してください。
